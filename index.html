prim·i·tive

adjective
being the first or earliest of the kind or in existence, especially in an early age of the world: primitive forms of life.

In the realm of computer science, a primitive is the simplest data type available to a programmer.

When I started to learn how to use computer programming to create images I gave myself a constraint, to use only “primitive” shapes. The language that I used, Processing, lists the point, line, arc, quadrilateral, rectangle, and ellipse as two dimensional primitives. I further refined my constraint to be one primitive type, the ellipse, as the sole building block for a series of programs. The goal of using this constraint was to not become distracted by all the tools and features suddenly available to me. Any visual complexity would come from the code that I wrote.

The Primitive series is an ongoing project. Some of the programs are interactive, the user can employ the mouse to alter the images.
